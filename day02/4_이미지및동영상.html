상대경로 vs 절대경로 1. 상대경로 : 현재 문서(위치) 기준으로 상대적인 경로 1. ./ 또는 생략 : 현재 위치 2. ../ : 현재 위치의 상위 위치로 1번 이동 3. ../../ : 현재 위치의 상위 위치로 2번 이동 4. ../폴더명/파일명 : 현재 위치의 상위 위치로 1번 이동후 폴더 이동 2. 절대경로 : 모든 경로
